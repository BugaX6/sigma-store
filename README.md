# 🛍️ SigmaStore

O **SigmaStore** é uma aplicação web de e-commerce focada no segmento de perfumaria, cosméticos e cremes masculinos e femininos[cite: 1]. O projeto simula uma experiência completa de loja digital, integrando rotas protegidas, gerenciamento de catálogo e autenticação de usuários[cite: 1].

## 🚀 Tecnologias Utilizadas

- **Angular 21**: Framework principal utilizado para a construção da SPA (Single Page Application), aproveitando os recursos modernos de componentes[cite: 1].
- **TypeScript**: Superset do JavaScript que garante tipagem estática e maior segurança no desenvolvimento[cite: 1].
- **JSON Server (Simulado)**: O arquivo `db.json` estrutura a base de dados do projeto, simulando uma API REST real para persistência e consumo de dados de usuários e produtos[cite: 1].
- **Vitest**: Framework de testes ágil de última geração configurado para a validação das unidades de código[cite: 1].

## ⚙️ Funcionalidades Principais

- **Autenticação e Controle de Acesso**: Sistema de login com usuários pré-definidos (incluindo perfis Administrativos e de clientes)[cite: 1].
- **Catálogo Dinâmico de Perfumaria**: Separação inteligente e renderização de perfumes e cremes filtrados por categorias masculinas e femininas[cite: 1].
- **Interface Responsiva**: Estrutura modular de componentes otimizada para navegação fluida[cite: 1].
